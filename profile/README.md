## 🌍 Olùkọ́ — Learn African Languages, the Interactive Way

**Olùkọ́** *(oh-loo-kaw)* is the **Duolingo for African languages** — designed to make learning both accessible and culturally meaningful.

### 🚀 What We're Building
Starting with the **Yoruba language**, Olùkọ́ offers:
- 🤖 **Real-time conversations with AI tutors**
- 📖 **Engaging reading exercises**
- 🎧 **Interactive listening practice**

### 🌱 Our Mission
By combining modern technology with deep cultural context, Olùkọ́ aims to:
- Preserve and promote African languages 🌍  
- Make language learning accessible to everyone 📚  
- Create immersive, authentic learning experiences ✨
